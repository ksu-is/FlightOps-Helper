#  FlightOps Helper

## Description
FlightOps Helper is a Python-based flight operations management system that allows users 
to track, manage, and update flight information through a clean and interactive dashboard. 
Inspired by real airline operations, this system organizes flight data including flight 
numbers, destinations, departure times, gate assignments, and live status updates — all 
in one place.

## The Problem It Solves
Managing flight operations manually is slow, scattered, and error-prone. FlightOps Helper 
centralizes all flight data, automatically flags gate conflicts, and makes status updates 
fast and efficient — no deleting and re-entering information required.

## Who It's For
- Airport and airline operations staff
- Aviation students and enthusiasts
- Anyone curious about how flight management works behind the scenes

## Features
- Add flights with details: flight number, destination, departure time, and status
- Assign gates to flights
- Automatic gate conflict detection (flags duplicate gate assignments at the same time)
- Update flight status easily (On Time / Delayed / Boarding / Cancelled)
- Clean and interactive dashboard built with Streamlit

## Tech Stack
- Python — core programming language
- Pandas — data handling and flight record management
- Streamlit — interactive dashboard and user interface
- SQLite — local database for storing flight information
